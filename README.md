# Fenotypefairy — luthfabushra.github.io

Static, GitHub Pages–friendly website (HTML/CSS + minimal JS).

## Publish (GitHub Pages user site)

1. Create a GitHub repository named **luthfabushra.github.io** (exact name).
2. Push this repo contents to the **main** branch.
3. In GitHub: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**

Once it’s deployed, the site will be available at:

- https://luthfabushra.github.io/

## Local preview

Use VS Code Live Server (recommended), or any static server.
